# Evidence Index

这里保存最新研究包的证据与验证材料。

- `source-ledger.md`：报告使用的来源总表。
- `claim-ledger.md`：关键 claim 与来源映射。
- `verify-report.md`：结构、PDF、来源可达性与章节完整性验证结果。
- `strawmap-research-gate-review.md`：独立 reviewer 的 gate review。
- `source-refresh-current.tsv`：本轮 122 个 URL 的 HTTP 可达性记录；120 OK、2 restricted、0 bad。
- `detailed-source-refresh.tsv`：上一轮 115 个 URL 的 HTTP 可达性记录；它不是逐条 claim 正确性的自动证明。
- `source-refresh-status.tsv`：早期来源刷新状态。
- `structure-source-check.txt`：报告数量、URL、坏链、章节检查摘要。
- `strawmap-current.html` / `strawmap-current.headers.txt`：整理时抓取的 Strawmap 页面快照。
- `REPORT_PACK-pdf.txt`：从 PDF 抽取出的文本，用于检查 PDF 内容是否完整。
